# Keep Hosting Auto Repair Services

Premium, mobile-first marketing website for Keep Hosting Auto Repair Services — KwaZulu-Natal, South Africa.

## Stack
Static site (HTML / CSS / vanilla JS). Zero build step.

## Local preview
Open `index.html` in a browser, or serve the folder with any static server.

## Deploy
Configured for Netlify via `netlify.toml`. Drag-and-drop the folder, or connect the GitHub repo.
